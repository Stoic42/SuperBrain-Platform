## 🤖 常用AI提示词

### 分析项目进展
```
请帮我分析AstroPager项目的最新提交记录，重点关注：
1. 开发方向和进度
2. 技术实现细节
3. 团队协作情况
4. 潜在问题或风险
5. 需要我关注的重点
```

### 更新项目跟踪
```
基于AstroPager项目的最新进展，请帮我更新项目跟踪文档：
1. 更新最近提交记录
2. 调整项目里程碑状态
3. 识别新的风险点
4. 更新当前任务列表
5. 调整下周计划
```

### 生成复盘分析
```
基于新的共学活动，请帮我生成复盘分析：
1. 活动目标达成情况
2. 团队表现分析
3. 技术培训效果
4. 发现的问题和挑战
5. 改进建议和行动计划
6. 下次活动准备事项
```

### 风险分析
```
检测到AstroPager项目风险事件，请帮我：
1. 分析风险类型和影响程度
2. 评估对项目进度的影响
3. 制定应对策略
4. 更新风险跟踪记录
5. 通知相关人员
6. 调整项目计划
```
