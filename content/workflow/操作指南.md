# SuperBrain平台操作指南

## 🎯 使用说明
本指南提供了SuperBrain平台的日常操作流程，基于AI原生创新工作流理念，包括何时做什么、如何操作，以及使用什么提示词让AI协助完成任务。

## 🤖 AI原生工作流核心理念
- **人机协作**: AI是思考伙伴，不是执行工具
- **上下文工程**: 丰富的上下文 = 精准的AI输出
- **反思迭代**: 实践-复盘-反思-迭代的完整闭环
- **过程导向**: 关注学习过程，而非仅看结果

## 🤖 AI原生工作流实践方法

### 1. 不插电创意梳理
**目标**: 让团队原生想法充分碰撞，形成高质量的初始上下文
**操作**:
- 不要急于查资料或问AI
- 先进行团队讨论，充分表达原始想法
- 记录讨论要点和分歧观点
- 形成清晰的初始文档

**AI提示词**:
```
请帮我整理团队讨论的要点，重点关注：
1. 核心问题和解决方案
2. 目标用户和痛点
3. 功能模块划分
4. 团队分歧点
5. 需要进一步澄清的问题

讨论记录：[粘贴团队讨论内容]
```

### 2. AI反向提问与上下文深化
**目标**: 让AI成为产品经理，通过提问澄清需求
**操作**:
- 将团队想法汇总输入给AI
- AI每提一个问题，团队快速讨论（2-3分钟）
- 记录不同观点，都告诉AI
- 完成6-8轮问答，让需求越来越清晰

**AI提示词**:
```
作为产品经理，请对以下项目想法进行需求澄清提问：

项目背景：[粘贴项目背景]
团队讨论要点：[粘贴讨论要点]
分歧点：[粘贴分歧点]

请提出5-8个关键问题，帮助团队澄清需求。每个问题请说明：
1. 为什么问这个问题
2. 希望得到什么信息
3. 对后续设计的影响

请一次只问一个问题，等待团队回答后再问下一个。
```

### 3. 个人筛选与集体讨论
**目标**: 体现"人的品味"，融合人类智慧优化AI方案
**操作**:
- 每人独立阅读AI输出，进行标注：
  - ✅ 认同的部分（绿色高亮）
  - ❓ 有疑问的部分（黄色高亮）
  - ❌ 不认同的部分（红色高亮）
- 团队讨论有分歧的部分
- 基于实际情况做筛选和决策

**AI提示词**:
```
基于以下团队标注和讨论，请帮我优化项目方案：

AI原始方案：[粘贴AI方案]
团队标注：
- 认同部分：[粘贴绿色标注内容]
- 疑问部分：[粘贴黄色标注内容]
- 不认同部分：[粘贴红色标注内容]

团队讨论结果：[粘贴讨论结果]

请基于团队实际情况，优化方案并说明调整理由。
```

## 📅 日常操作流程

#### 1. 检查AstroPager项目进展
