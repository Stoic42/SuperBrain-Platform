## 🚀 快速开始

### 每日必用提示词

#### 1. 晨检项目进展
```
请帮我分析[项目名称]的最新进展，重点关注：
1. 开发方向和进度
2. 技术实现细节  
3. 团队协作情况
4. 潜在问题或风险
5. 需要我关注的重点

最新提交记录：[粘贴git log输出]
```

#### 2. 团队状态评估
```
作为领航员，请帮我评估团队当前状态：
1. 团队健康度评估
2. 潜在问题识别
3. 需要关注的重点
4. 建议的干预措施
5. 下周重点关注事项

团队信息：[描述团队情况]
```

#### 3. 复盘会议准备
```
请帮我准备本周团队复盘会议：
1. 复盘会议议程
2. 关键问题清单
3. 引导性问题
4. 讨论重点
5. 行动项模板

本周数据：[描述本周情况]
```

## 🔧 工具使用技巧

### 提示词优化技巧
1. **上下文丰富**: 提供充分的背景信息
2. **目标明确**: 清晰说明期望的输出
3. **结构标准**: 使用统一的格式和结构
4. **迭代优化**: 根据AI输出调整提示词

### 效果评估方法
1. **输出质量**: 检查AI输出的准确性和完整性
2. **执行效率**: 评估提示词的使用效率
3. **适用性**: 判断提示词是否适合当前场景
4. **改进空间**: 识别提示词的改进机会

### 最佳实践
1. **个性化调整**: 根据具体情况调整提示词内容
2. **组合使用**: 可以组合多个提示词完成复杂任务
3. **模板定制**: 基于模板创建个性化提示词
4. **经验分享**: 与团队分享有效的提示词
