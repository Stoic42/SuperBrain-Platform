# 超脑黑客松综合管理平台 - 用户指南

## 📖 平台简介

超脑黑客松综合管理平台是一个专为超脑AI线上黑客松1.5项目设计的综合管理工具，集成了项目管理、团队协作、进度跟踪、资源管理等多个功能模块。

## 🚀 快速开始

### 1. 打开平台
在浏览器中打开 `超脑黑客松综合管理平台.html` 文件即可开始使用。

### 2. 导航说明
平台顶部有7个主要功能标签页：
- **📊 项目总览**: 查看项目整体状态和关键指标
- **👥 团队管理**: 管理团队成员信息和状态
- **📈 进展跟踪**: 跟踪项目进度和任务完成情况
- **🏆 项目挑战**: 查看项目挑战和奖励信息
- **🤖 AI工作流**: 了解AI原生工作流程
- **📚 资源中心**: 访问项目相关文档和资源
- **📊 数据分析**: 查看项目数据分析和对比

### 3. 快捷键支持
- `Ctrl + 1`: 切换到项目总览
- `Ctrl + 2`: 切换到团队管理
- `Ctrl + 3`: 切换到进展跟踪
- `Ctrl + 4`: 切换到项目挑战
- `Ctrl + 5`: 切换到AI工作流
- `Ctrl + 6`: 切换到资源中心
- `Ctrl + 7`: 切换到数据分析

## 📊 项目总览

### 功能说明
项目总览页面显示项目的整体状态，包括：
- 项目基本信息和描述
- 关键指标统计（学员数量、项目数量、完成率等）
- 紧急提醒和警告信息
- 项目核心理念展示

### 关键指标
- **学员总数**: 当前参与项目的学员数量
- **项目组**: 项目组的数量
- **导师团队**: 导师和领航员的数量
- **作业完成率**: 学员作业完成的比例
- **环境配置率**: 技术环境配置完成的比例
- **整体进度**: 项目整体完成进度

## 🤖 AI工作流

### 5步工作流程
1. **不插电创意梳理（25分钟）**: 团队原生想法碰撞
2. **AI反向提问与上下文深化（30分钟）**: AI作为产品经理提问
3. **AI生成初步架构方案（25分钟）**: AI生成多维度建议
4. **集体讨论与人类优化（35分钟）**: 融合人类智慧优化方案
5. **复盘与总结（5分钟）**: 提炼经验和理解核心

## 🛠️ 故障排除

### 常见问题

#### 1. 页面无法正常显示
- 检查浏览器是否支持HTML5和CSS3
- 确保JavaScript已启用
- 尝试刷新页面

#### 2. 链接无法打开
- 检查文件路径是否正确
- 确保目标文件存在
- 检查文件权限

#### 3. 数据不更新
- 检查网络连接
- 刷新页面重新加载数据
- 检查数据文件是否存在
