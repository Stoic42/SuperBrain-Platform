# 超脑黑客松综合管理平台

## 🎯 平台概述

这是一个为超脑AI线上黑客松1.5项目设计的综合管理平台，整合了项目管理、团队协作、进度跟踪、资源管理等多个功能模块。

## 🚀 快速开始

### 环境要求

- Node.js (建议 v18.x 或更高版本)
- pnpm (或 npm/yarn)

### 本地安装与启动

1.  **克隆你的 Fork 仓库**
    ```bash
    git clone https://github.com/YOUR_USERNAME/SuperBrain-Platform.git
    cd SuperBrain-Platform
    ```

2.  **安装依赖**
    ```bash
    pnpm install
    ```

3.  **环境配置**
    - 复制环境变量模板文件：
      ```bash
      cp .env.example .env.local
      ```
    - 根据 `.env.local` 文件中的注释，填写必要的配置（如数据库地址、API密钥等）。

4.  **启动开发服务器**
    ```bash
    pnpm dev
    ```
    现在，在浏览器中打开 `http://localhost:3000` 即可看到本地运行的平台。

## 🤝 如何贡献

我们采用标准的 **Fork & Pull Request** 工作流。

### 贡献流程

1.  **Fork 仓库**
    - 如果你还未操作，请先 Fork [官方主仓库](https://github.com/ton-ton-fun/SuperBrain-Platform) 到你自己的 GitHub 账户。

2.  **创建新分支**
    - 从 `main` 分支创建一个描述性功能分支。
      ```bash
      # 确保你的 main 分支与官方仓库保持同步
      git pull upstream main
      
      # 创建你的新功能分支
      git checkout -b feat/your-cool-feature
      ```
    - **分支命名规范**:
        - `feat/feature-name`: 新功能
        - `fix/bug-name`: Bug 修复
        - `docs/update-readme`: 文档修改

3.  **提交你的代码**
    - 编写代码，并在本地充分测试。
    - 使用清晰的 commit message 提交你的改动。
      ```bash
      git add .
      git commit -m "feat: add user login functionality"
      git push origin feat/your-cool-feature
      ```

4.  **创建 Pull Request (PR)**
    - 将你的功能分支推送到你的 Fork 仓库后，在 GitHub 上发起一个 Pull Request 到官方仓库的 `main` 分支。
    - 在 PR 描述中清晰地说明你做了什么、解决了什么问题。

我们会在收到 PR 后尽快进行代码审查。感谢你的贡献！
